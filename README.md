# Unique Handwriting Classes — Website

A modern, SEO-optimised marketing website for **Unique Handwriting Classes**, a handwriting coaching institute based in Sanath Nagar, Hyderabad, Telangana.

![Website Preview](./public/images/website-preview.png)

---

## About

Unique Handwriting Classes helps children improve neatness, cursive writing, spacing, letter formation, grip, and writing confidence through guided practice and personal attention. This website serves as their primary online presence — showcasing their program, results, location, and free YouTube resources.

## Getting Started

### Prerequisites

- Node.js 18+
- npm or yarn

### Installation

```bash
# Clone the repository
git clone <repo-url>
cd unique-handwriting-website

# Install dependencies
npm install

# Start the development server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

### Build for Production

```bash
npm run build
npm run start
```
